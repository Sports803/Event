How it works

prog.txt — A pipe-delimited text file listing today's matches with embed URLs. Format: HH:MM Sport: TeamA vs TeamB | embed_url1 | embed_url2

ddkaqiu.com — Chinese sports streaming site (replaced ddkanqu.com). Homepage lists live/upcoming matches with links to /zhibo/{id}.html. Chinese team names are translated via built-in mapping.

CORS proxies — Both sources use corsproxy.io and api.allorigins.win for browser-side fetching since these sites don't set CORS headers.

Team matching — Team names are normalized (lowercase, stripped of accents, club abbreviations removed) and matched case-insensitively across both sources.

Theme field reference

The Blogger theme reads events from s803config/todaysMatches. Key fields the theme uses:

kickoff — ISO date string, used for countdown timers.

homeName / awayName / homeLogo / awayLogo — team display.

leagueName / leagueEmoji / leagueId — competition badge.

channels — array of {label, src} objects for stream buttons.

channelKey / channelName — server-side channel resolution via livetv/channels.

statusType — STATUS_SCHEDULED, STATUS_LIVE, STATUS_FINAL.

score / scoreHome / scoreAway — live score placeholders.

category — one of: football, basketball, nfl, hockey, baseball, f1, nascar, motorsport, boxing, mma, cycling, cricket, other.

duration — minutes, controls the live window length.

isRacing — boolean, hides away-team markup for single-entrant events.
